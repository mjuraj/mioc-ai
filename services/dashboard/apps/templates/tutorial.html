Getting started

Managing users

Users:
Here you can view and add users. Obligatory fields are marked with asterisks*

Managers:
Here you view and edit contact persons for the users. For testing, the phone number can be fake, but email has to be correct for scheduling calls in the calendar.

Managing the assistant

Assistants:
Select an assistant and edit prompts to get different behavior or tone of voice. Add files to assistant's knowledge base by selecting them at the bottom under Knowledge Base and save.

Knowledge: Drag and drop files in the Upload file tab and click Submit. After that you can find them in the tab Knowledge base. In the Knowledge base, you can open files to edit them, just press save. When you want to add the uploaded file to the Knowledge base, select it and pick action "Add to knowledge base" in the dropdown below. Refresh, and when the status of the file changes to "ADDED AS KNOWLEDGE", you can add it to your assistant.
